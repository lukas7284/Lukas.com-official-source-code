how to set lukas xp up.

1. download the iso file to use on a virtual machine.

click this non loading image to download it.

2. go to your hypervisor (virtual machine software) and click new. they will guide you through the rest of the process.

3. load the iso file. you may see it in the new machine setup or right when you boot the virtual machine up.

when you see this, load the iso file that is provided on step 1.

4. when you load it up, you should see a blue, bios like menu. select start installation by pressing enter.

you will see the terms, press the f8 key to agree. if pressing f8 doesn't work, for some computers, you will need to press fn+ f8.

then you will see 2 options for partitioning, select the first option.

 

next you will see a list of options for formatting. select any of them.

 

after you do that you should see a progress bar. wait for everything to finish

 

5. the setup:

wait a second for windows to loom for devices.

next you will see some regional settings. unless you from china or something, leave them alone

next, enter your name and your desired computer name then click next. Now set the time.

then, wait for windows to install more things. then you will be asked about workgroup settings. just say no and type anything for the workgroup name.

you will receive an error about the workgroup being invalid. just click yes. wait again for more of the installation. and the finalizing.

then, the installer will try to improve the resolution but It probably wont do much.

then wait.

you will then see a guided setup of lukas xp followed by some amazing music if the sound works.

when you see a message asking you what type of internet you are using just click connect through a local or home network.

after you set everything up, windows will start and will show a command prompt window. just close it unless there is something you need to do there.

you will see a list of programs and other things that windows will set up.

you should see this. if you see this, you have successfully set up lukas xp.

 

if you still feel lost, click start and tour windows xp.
